# 덱 빌딩 디펜스 (Deck Building Defense)

직관적인 모바일 게임으로 구현된 덱 빌딩 디펜스 게임입니다.

## 🚀 게임 실행 명령어

### 브라우저에서 실행 (가장 빠름)
```bash
flutter run -d chrome
```

### iOS 시뮬레이터에서 실행
```bash
flutter run -d "iPhone 15 Pro"
```

### 모든 디바이스에서 실행
```bash
flutter run -d all
```

### 기본 실행 (안드로이드)
```bash
flutter run
```

## 🎮 게임 소개

스타크래프트 유즈맵 "덱 빌딩 디펜스"의 규칙을 기반으로 한 모바일 게임입니다. 도미니언 스타일의 덱 빌딩과 타워 디펜스 요소를 결합하여 전략적인 게임플레이를 제공합니다.

## 🚀 주요 기능

### 게임 시스템
- **30라운드 생존**: 각 라운드 30초, 최종 라운드는 45초
- **보스 라운드**: 5/10/15/20/25/30 라운드마다 특별한 보스 등장
- **덱 관리**: 매 턴 5장 드로우, 최대 32장 보유
- **마나 시스템**: 매 턴 마나 2 제공, 카드 구매시 마나 1 소모

### 카드 시스템
- **자원 카드**: 미네랄 제공 (기본: 1, 고급: 3)
- **유닛 카드**: 방어력 무시 공격, 공업 레벨에 따라 20% 추가 피해
- **효과 카드**: 추가 드로우, 마나 부스트 등 특수 효과
- **마법 카드**: 치유, 공업 강화 등 유지형 카드

### 전투 시스템
- **3가지 적 타입**: 
  - 물량형: 다수 출현
  - 방어형: 높은 방어력
  - 쉴드형: 높은 HP + 쉬드
- **공업 시스템**: 유닛 공격력 20% 증가
- **실시간 전투**: 제한시간 내 적 처치

## 📱 기술 스펙

- **플랫폼**: Flutter (iOS/Android 크로스플랫폼)
- **UI**: 가로 화면 최적화, 다크 테마
- **애니메이션**: Flutter Animate 활용 부드러운 애니메이션
- **사운드**: 게임 상황에 맞는 효과음 시스템
- **상태관리**: Provider 패턴
- **저장소**: Hive (로컬 데이터)

## 🎯 게임플레이

1. **준비 단계**: 매 턴 시작시 마나 2, 카드 5장 드로우
2. **구매 단계**: 상점에서 카드 구매 (마나 1)
3. **전투 단계**: 라운드 시작하여 적 처치
4. **정리 단계**: 손패 정리 후 다음 턴

## 🏆 승리 조건

- 30라운드 모든 적을 처치하고 생존
- 최종 보스 처치

## ❌ 패배 조건

- 플레이어 HP 0 이하
- 라운드 종료시 적 51마리 이상 생존
- 보스 2기 이상이 필드에 남아있을 때
