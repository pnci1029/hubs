# 덱 빌딩 디펜스 — 게임 디자인 문서

> 원작: 스타크래프트 유즈맵 "덱 빌딩 디펜스" (도미니언 메커니즘 기반)
> 목표: 1인 플레이 웹 게임으로 재구성

---

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 덱 빌딩 + 웨이브 디펜스 |
| 플레이어 | 1인 |
| 목표 | 30라운드 동안 기지(HP)를 지키며 생존 |
| 패배 조건 1 | 기지 HP가 0 이하 |
| 패배 조건 2 | 누적 탈출 적군 50마리 초과 |
| 승리 조건 | 30라운드 클리어 |

---

## 2. 기본 흐름 (턴 구조)

```
[라운드 시작 — 준비 단계]
    ↓
① 덱에서 카드 5장 드로우 + 마나 2 획득
    ↓
② 플레이어 턴
   - 자원 카드 플레이 → 미네랄 획득
   - 효과 카드 사용 → 드로우·미네랄·마나·공업 등 특수 효과
   - 마법 카드 사용 → 별도 슬롯에서 언제든지 사용 가능 (최대 3장 보유)
   - 미네랄 + 마나 1 소비 → 상점에서 카드 구매
    ↓
③ 전투 단계 시작
   - 유닛 카드는 자동 공격 (덱·손패·버림더미·지하실 전부 참여)
   - 효과·마법 카드 수동 사용 가능
   - 처치 못한 적이 기지를 공격 (일반 1 피해, 보스 3 피해)
    ↓
④ 턴 종료
   - 손패 + 플레이한 카드 전부 버림 더미로
   - 덱이 비면 버림 더미 셔플 → 새 덱
    ↓
[다음 라운드]
```

---

## 3. 카드 타입 (8종)

| 타입 | 색상 | 설명 |
|------|------|------|
| `resource` | 청색 | 플레이 시 미네랄 획득. 덱 순환의 기초이자 후반 압축 대상 |
| `unit` | 적색 | 전투 시 자동 공격. 화력 중심 |
| `effect` | 보라 | 드로우·마나·미네랄·공업 등 다양한 즉발 효과. 덱 순환에 참여 |
| `resource_effect` | 청록 | 자원 + 효과 겸용 |
| `unit_effect` | 분홍 | 유닛 자동 공격 + 손패 효과 겸용 |
| `combo` | 시안 | 자원 + 유닛 겸용 |
| `trash` | 회색 | 플레이 시 미네랄 감소. 덱의 찌꺼기 |
| `magic` | 금색 | 덱 순환과 무관한 별도 마법 슬롯. 최대 3장. 사용 즉시 소멸 |

## 4. 핵심 메커니즘

### 미네랄 & 마나

| 자원 | 획득 방법 | 용도 |
|------|-----------|------|
| 미네랄 ⬡ | 자원 카드 플레이 | 카드 구매 비용 |
| 마나 | 라운드 시작 시 +2 (기본) | 카드 구매 시 1 소모 / 일부 카드 직접 사용 |

> 카드 구매 = 미네랄(카드 코스트) + 마나 1 소모

---

### 덱 순환

```
드로우 → 플레이 → 버림 더미 → 셔플 → 다시 드로우
```
- 덱이 비면 버림 더미를 셔플해서 새 덱으로 재활용
- SCV 같은 저가 카드가 많을수록 순환이 느려짐
- → **파괴(트래쉬) 카드**로 덱을 압축하는 것이 핵심 전략

---

### 지하실 (Trash Zone)

- 특정 카드 효과로 손패·덱의 카드를 **영구 파괴** 가능
- 파괴된 카드는 **지하실**에 보관 → 덱 순환에서 제외 (다시 드로우되지 않음)
- **단, 전투 시 자동 공격에는 계속 참여** → 지하실에 쌓인 유닛도 화력에 기여
- 덱뷰어의 🏚 지하실 탭에서 확인 가능

> 전략 포인트: SCV·잉여 같은 약한 카드를 지하실로 보내면 덱이 압축되어 순환이 빨라지고, 동시에 SCV의 자동 공격 기여도 유지됨

---

### 마법 카드 (Magic Zone)

- 구매 즉시 **마법 슬롯**에 보관 (버림 더미로 가지 않음)
- 덱 순환과 무관 — 드로우되지 않음
- **언제든지** 클릭으로 사용 가능 (준비·전투 단계 무관)
- 사용 즉시 소멸 (지하실에도 가지 않음)
- **최대 3장** 보유 가능 (꽉 차면 상점에서 매진)

| 카드 | 코스트 | 효과 | 사용 가능 단계 |
|------|--------|------|--------------|
| 🔮 드로우 마법 | 4 | 즉시 카드 +3 드로우 | 준비·전투 |
| ✨ 미네랄 마법 | 5 | 즉시 미네랄 +5 | 준비·전투 |
| 💥 공격 마법 | 6 | 전체 적에게 즉시 5 피해 | **전투 중 전용** |
| 💊 회복 마법 | 7 | 기지 HP +5 회복 | 준비·전투 |

---

### 필드 유닛 시스템

- 전투 시 **덱 + 손패 + 버림 더미 + 지하실** 전체의 카드가 필드 유닛으로 배치
- 각 카드는 고유 공격력(atkDmg), 공격 속도(atkSpd), 사거리(atkRange), 공격 타입을 가짐
- 공업(bonusPower) 1당 전체 공격력 +10%

| 공격 타입 | 설명 |
|-----------|------|
| `normal` | 방어력 감소 후 피해 |
| `piercing` | 방어력 무시 |
| `explosive` | 방어력 감소 + 방사 피해 |
| `splash` | 방사 피해 |

---

### 마린 토큰

- THREE 카드, COWSHED 등 특정 카드가 **마린 토큰** 생성
- 토큰은 덱에 포함되지 않고 전투 시 필드 최우선 배치
- 공격력 10, 관통형, 사거리 150

## 6. 적 (Enemy) 웨이브

### 적 종류

| 적 | HP | 방어력 | 보호막 | 속도 | 타입 |
|----|-----|--------|--------|------|------|
| 🐛 저글링 | 50 | 0 | 0 | 빠름 | 경장 |
| 🐍 히드라리스크 | 200 | 1 | 0 | 중간 | 중장 |
| 🦏 울트라리스크 | 600 | 3 | 0 | 느림 | 중장갑 |
| 💀 군주 (보스) | 2000 | 2 | 300 | 매우 느림 | 중장갑 |

- 라운드가 올라갈수록 HP 강화 (라운드당 +6%)
- 보스 등장: 5·10·15·20·25·30라운드
- 처치 못한 적 → 기지 공격 (일반 1 피해, 보스 3 피해) + 누적 탈출 카운트 증가

### 웨이브 구성

| 라운드 | 구성 |
|--------|------|
| 1–4 | 저글링 위주 |
| 5–9 | 저글링 65% + 히드라 35% |
| 10–19 | 저글링·히드라·울트라 혼합 |
| 20+ | 히드라·울트라 위주 |
| 5·10·15·20·25·30 | +보스 1마리 추가 |

## 9. 전략 가이드

| 단계 | 핵심 전략 |
|------|-----------|
| 초반 (1–5) | SCV를 파괴 카드로 지하실 이송 → 덱 압축. 마린 추가 확보 |
| 중반 (6–15) | 드론·프로브 구매로 미네랄 수입 확대. 효과 카드 콤보 발굴. 마법 카드 1–2장 확보 |
| 후반 (16–30) | 강력한 유닛 카드 + 공업 스택. 마법 카드 위기 대응용 비축. 보스 라운드 대비 |

> **덱 압축 핵심**: 지하실에 보내는 카드는 순환에서 빠지지만 자동 공격에는 계속 참여. 강한 유닛은 지하실에 보내지 말 것.
